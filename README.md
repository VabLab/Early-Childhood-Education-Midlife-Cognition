# Is early childhood education associated with better midlife cognition, especially for children facing socioeconomic marginalization?

By Whitney Wells, Jillian Hebert, Chloe W. Eng, Catherine dP Duarte, Anusha M. Vable

Link to paper: TBD

# Repository Content
- `README.md`: This file with an explanation of this repository.
- `1. Cognition Operationalization.R`: R code for operationalizing midlife cognition variables and composite measures.
- `2. Clean Exposure and Covars.R`: R code for cleaning NLSY exposure and covariate data.
- `3. Merge and Impute.R`: R code for merging in outcome data and performing imputation.
- `4. Post Imp Cleaning.R`: R code for cleaning imputed dataset.
- `5. Analysis.R`: R code for all analysis for manuscript.
